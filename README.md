# Qmovies

### A one-stop video streaming platform for you to watch TV shows, movies, animation from all around the world! With wide variety of exclusive movies, you can choose multi-language subtitles.

![](https://i.imgur.com/waxVImv.png)

### Technologies which I use in this project:

- ReactJS
- React router dom
- Swiper (Create Slider)
- React Loading Skeleton
- React Lazy Load Image Component
- React Intersection Observer ( Detect when an element enters or leaves the viewport)
- React Infinite Scroll Component (Infinite scrolling)
- React HLS Player (HLS live stream player)
